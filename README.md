如何在cmd（命令提示符）中打开指定文件夹路径
方法一（最快最简单）
打开指定的文件夹，在路径栏里输入“cmd”，回车，就进入控制台了。默认路径就是指定文件夹的路径。

在cmd命令行中打开test.html所在文件夹然后使用命令wkhtmltopdf test.html test.pdf就会发现当前文件夹下出现了test.pdf的文件
